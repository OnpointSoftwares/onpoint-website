Purchase Order Workflow

Creating a Purchase Order

  1. Navigate to Inventory > Purchase Orders
  2. Click New Purchase Order
  3. Select supplier from dropdown
  4. Add products manually or from low stock alerts
  5. Set quantities and expected delivery dates
  6. Review totals and submit order

Receiving Orders

  1. Go to Purchase Orders > Pending
  2. Select order to receive
  3. Scan or manually enter received quantities
  4. Handle discrepancies if any
  5. Update costs if different from ordered
  6. Complete receipt to update inventory
Smart Features: Auto-reorder points, supplier performance tracking, and cost analysis help optimize your procurement process.

User Guide

Processing a Sale

  1. Click New Sale from dashboard or Sales module
  2. Add products by scanning barcodes or searching by name
  3. Apply discounts/promotions if applicable
  4. Select payment method (Cash, Card, Mobile Pay)
  5. Mark payment status and Complete transaction
  6. Print thermal receipt (80mm) or email as needed
  7. Dashboard KPIs refresh automatically after successful sale
Quick Tip: Use barcode scanner for faster product lookup during busy periods.

Managing Inventory

Adding a New Product

  1. Go to Inventory > Products
  2. Click Add Product
  3. Fill product details (name, SKU, price, etc.)
  4. Set initial stock level
  5. Upload product images if available
  6. Click Save

Updating Stock Levels

  1. Go to Inventory > Stock Adjustment
  2. Select product to update
  3. Enter new quantity or adjust by amount
  4. Add note explaining adjustment
  5. Click Update Stock

Creating a Quotation

  1. Open Quotations module and click New Quotation
  2. Enter customer and company details (logo appears on print)
  3. Add items and quantities, then Save
  4. Click Print to generate professional quotation
  5. Convert to sale later via Convert to Sale

Printing Setup

  • Connect thermal printer (80mm recommended)
  • Set as default system printer
  • Configure print behavior in Settings
  • Test with sample receipt

Troubleshooting

Possible Causes:

  • Java is not installed or not in system PATH
  • Insufficient system resources
  • Corrupted installation files

Solutions:

  1. Verify Java: java -version
  2. Download latest JRE from Eclipse Temurin
  3. Try reinstalling the application

Possible Causes:

  • Insufficient file permissions
  • Database file locked by another process
  • Corrupted database file

Solutions:

  1. Ensure write permissions to installation directory
  2. Restart computer to clear processes
  3. Restore from recent backup if available

If install-windows.ps1 is blocked by execution policy:

Frequently Asked Questions

How do I update the application?

To update BizFlow POS:

  1. Download the latest version
  2. Back up your database (pos.db)
  3. Install new version over existing
  4. Restore backup if necessary

Can I use BizFlow POS on multiple computers?

Yes, you can install on multiple computers. However, ensure only one instance accesses the database file at a time. For multi-user access, consider a centralized database server.

How do I back up my data?

  1. Close the application
  2. Locate pos.db in installation directory
  3. Copy to secure backup location

Set up automated backups with scheduled tasks or cron jobs.

Where is my data stored?

Windows: C:\Program Files\BizFlow POS\
Linux: /opt/bizflow-pos/
Database: pos.db in install folder

How do I change currency and tax settings?

Go to Settings → System Settings to configure:

  • Default currency (e.g., KES)
  • Tax rates and calculations
  • Regional formatting preferences

Does BizFlow POS work offline?

Yes! The application runs completely locally with SQLite database. Internet is only needed for updates or optional cloud backups.

I forgot the admin password. How can I reset it?

If you have another admin account, reset via Settings → User Management. Otherwise, contact your system administrator for backup restoration or secure reset procedures.
